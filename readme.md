## Install:

```
npm install -g dat-next
```

## Usage

### Share a directory:

```
dat-next <dir> [options]
    --temp,-t     keep sleep data in memory
    --watch, -w   watch folder for changes + auto import
```

### Download `key` to `dir`:

```
dat-next <key> <dir> [options]
    --live     keep running + update changes
```
